# Rainbow Creations

A website for Rainbow Creations.

## Instructions

### Dev dependencies

* `node`
* `yarn`
* `nodemon`
* `gulp`

### Development server

Before running server for first time run `gulp` to build files.

Start development server with `yarn dev`.

### Deploy

Build the assets by running `gulp`.
